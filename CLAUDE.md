# IronX Fitness — Gym Website Project

## Project Overview
Recreating a gym website design (IronX Gym & Fitness theme, referenced from Behance screenshot at /reference/) using plain HTML, CSS, and JavaScript. No frameworks.

## Reference
Visual reference screenshot: reference/ironx-screenshot.png
Note: This is a visual reference only. Do not use any Behance.net CSS/design tokens — those belong to Behance's own site, not the actual theme.

## Design System
- Background: dark navy/black (#0d0d0d, #111111)
- Text: white for headings, light gray (#a0a0a0) for body/secondary text
- Accent color: blue (used for the floating action button and links)
- Font: clean sans-serif (e.g., Inter, Poppins, or system-ui)
- Buttons: rounded corners, subtle hover states (lighten background or add underline)
- Section spacing: generous vertical padding (80-120px between sections)
- Images: full-bleed or card-style with rounded corners, dark overlay for text-on-image sections

## Page Structure (in order)
1. Navbar — logo left, nav links center, "Join Now" button right
2. Hero — "Power. Strength. Cardio." heading, subtext, hero image right, 2 feature badges
3. "More Than a Gym" — 3 stacked cards (Functional Training, Cardio Area, Fitness & Pilates)
4. "Our Expert Coaches" — coach name list + 2 coach photos + stats row (150+ / 34 / 240)
5. Gallery strip — 3 full-bleed images with captions
6. "Become a Member" CTA — centered heading + Join Now button
7. Footer — 4 columns (logo/tagline, quick links, contact, social + newsletter)

## File Structure
- index.html — all page markup
- style.css — all styling
- script.js — any interactivity (mobile menu toggle, smooth scroll, etc.)
- images/ — placeholder image folder

## Rules
- Fully responsive — mobile-first, stack to single column under 768px
- Keep code clean and commented
- Use placeholder image paths like "images/hero-gym.jpg" — I'll swap real images in later
- Match the dark, premium gym-brand aesthetic from the reference screenshot
